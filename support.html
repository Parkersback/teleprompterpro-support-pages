欢迎使用 TeleprompterPro。

如果你在使用中遇到问题，可以先检查以下项目：

  • 请确认已授予相机权限，以便显示实时预览。

English

Welcome to TeleprompterPro.

If you run into issues, please check the following first:

  • Make sure camera access is enabled for live preview.
  • Make sure microphone access is enabled for audio recording.
  • If videos are not saving to Photos, check photo library add permission.
  • If the app language does not switch, check your iPhone system language or per-app language setting.

For support or feedback, contact: b51514029@gmail.com
